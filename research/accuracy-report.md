## 常見錯誤模式分析

### 1. MetaMask 辨識困難（高頻錯誤）
在三頁文件中，「MetaMask」這個英文專有名詞反覆被誤認：
- MetaMask → Metalask（M→l，出現3次）
- MetaMask → Metabask（M→b，出現1次）
- MetaMask → MetaNask（M→N，出現2次）

原因推測：手寫或印刷體的大寫「M」在小字體下容易與「l」「b」「N」混淆。

### 2. 正簡體混淆
多處將正體字誤認為簡體字：
- 別 → 别
- 戶 → 户
- 決 → 决

### 3. 表格欄位順序錯亂
表格式文件中，OCR 在處理跨欄位讀取時容易出錯：
- 「姓名」→「名姓」（左右欄位順序反了）
- 「性別 女」的「女」被移到上一行
- 經濟狀況選項順序被打亂

### 4. 問答標記「問」「答」字常被漏掉
三份文件中，共有約 8 處「問」或「答」字被漏掉。這些字通常在表格的左側邊欄，字體較大或有框線，OCR 可能將其視為表格裝飾而非正文。

### 5. 相似字混淆
- 間 → 問
- 述 → 訴
- 大 → 太
- 案 → 業
- 載 → 戴
- 址 → 垃
- 因 → 困

這些都是筆畫結構高度相似的字，在掃描品質不佳時容易出錯。
